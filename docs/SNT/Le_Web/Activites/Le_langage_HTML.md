# Activité 2 - Le langage HTML <i class="fa-brands fa-html5" style="color: #e54d26;"></i>

## 1. Présentation du langage HTML

Le HTML est un langage de programmation et d’écriture des pages Web. Il a été inventé par Tim Berners Lee en 1989 et est toujours utilisé actuellement même s’il a connu des évolutions (la version actuellement utilisée est la version 5 du HTML, notée HTML5, qui permet de lire des vidéos directement dans les pages web).

Une présentation plus détaillée du langage `HTML` est disponible sur [cette page](../Ressources/Presentation_HTML.md).

1. Ouvrir une nouvelle page sur un navigateur (Mozilla Firefox ou Google Chrome)

2. Ouvrir un nouvel onglet et aller sur le site de l’ENT du lycée : [Site du lycée polyvalent Val de Lys d'Estaires](https://lpo-du-val-de-lys-

1. Se connecter à l’ENT (avec ses identifiants élèves), et aller sur l’application CAPYTALE

2. Aller dans la rubrique “Mes Activités” et entrer le numéro d’exercice suivant : **4b8c-3937375**

3. Modifier la page Web présentée pour créer une page faisant apparaitre :

    * Un grand titre qui contiendra votre nom, prénom et classe. (/1)  
    * Une liste avec vos points forts. (/1)  
    * Une autre liste avec vos points à travailler. (/1)  
    * Une dernière avec vos matières préférées. (/1)  
    * Un lien vers l'établissement scolaire fréquenté l'an dernier. (/2)
