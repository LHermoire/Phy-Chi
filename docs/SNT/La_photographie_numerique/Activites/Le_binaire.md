# Activité 1 - Le langage binaire

1. Se connecter à l’ENT (avec ses identifiants élèves), et aller sur l’application CAPYTALE.

2. Aller dans la rubrique “Mes Activités” et entrer le numéro d’exercice suivant : **dac0-4735508**

3. Répondre aux différentes questions ci-dessous. En cas de besoin, voici des liens d'aide :
    * Lien vers le cours sur le binaire : [Cliquer ici](../Ressources/Le_binaire.md)
    * Lien vers la vidéo Youtube expliquant la conversion entre binaire et décimal : [Cliquer ici](https://ladigitale.dev/digiview/#/v/62bd73b07ced1)

## Exercice 1 : Informations codées par des bits

Des élèves construisent une station météo et veulent enregistrer les températures positives de 0 à 33°C.

1. Combien d'informations peuvent être codées par **1 bit** ?

2. Combien d’informations peuvent être codées par **3 bits** ?

3. Combien d’informations peuvent être codées par **64 bits** ?

4. **Combien de bit(s)** faut-il pour coder les températures ?

5. Un élève pense qu’il faille coder les températures sur **8 bits**. Jusque quelle température peuvent-ils coder ?

## Exercice 2 : Convertir un nombre décimal en nombre binaire

2. Convertir le nombre **68** en langage binaire.

3. Convertir le nombre **34** en langage binaire.

4. Convertir le nombre **93** en langage binaire.

5. Convertir le nombre **42** en langage binaire.

6. Convertir le nombre **317** en langage binaire.

## Exercice 3 : Convertir un nombre binaire en nombre décimal

1. Convertir le code binaire  **100** en nombre décimal.

2. Convertir le code binaire  **1110** en nombre décimal.

3. Convertir le code binaire  **110** en nombre décimal.

4. Convertir le code binaire  **1010** en nombre décimal.

5. Convertir le code binaire  **1100010** en nombre décimal.

6. Convertir le code binaire  **101010** en nombre décimal.
